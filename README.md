#  Assignment 2 - Python Control Flow

##  Description

This assignment demonstrates the use of Python control flow concepts including:

1. if / elif / else conditions
2. for loops and while loops
3. break and continue statements
4. basic input handling

The goal is to build simple order processing utilities using only conditionals and loops.

##  How to Run

1. Open VS Code or terminal
2. Navigate to the project folder
3. Run each file separately:

```bash
python task1.py
python task2.py
python task3.py
python task5.py
```


##  Tasks Overview

###  Task 1: Discount Rules

1. Takes user input for order amount
2.  Applies discount based on conditions
3.  Calculates final amount with optional tax
4.  Handles invalid input


###  Task 2: Process Multiple Orders

1.  Uses a list of order amounts
2.  Applies discount rules using a for loop
3.  Prints summary (order → discount → final amount)
4.  Calculates total revenue
5.  Counts number of discounted orders


### Task 3: User Menu (while loop)

1.  Menu-driven program using while loop
2.  Add orders dynamically
3.  Show all orders and total
4.  Uses break and continue for control flow


###  Task 5: Loop Control

1.  Iterates through daily sales data
2.  Uses break for corrupted data (-1)
3.  Uses continue for zero sales
4.  Calculates running and final total

##  Important Notes

1.  Only basic control flow concepts are used
2.  No functions, classes, or external libraries are included (as per assignment rules)

## Key Features

1.  Clean and simple logic
2.  Proper use of loops and conditions
3.  Error handling for user input
4.  Use of break and continue statements
